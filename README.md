## Installation

### Source

``` go get -u https://github.com/jon20/doku```

## Usage

|panel  |operation  |key  |
|---|---|---|
|all  |change panel  |Tab  |
|all  |quit  |Ctrl + q  |
|Container List  |Container start  | r|
|Container List  |Container stop  | s|
